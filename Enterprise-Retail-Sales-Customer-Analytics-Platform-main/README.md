# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The objective is to understand customer purchase patterns, identify key trends, and generate business insights that can help improve sales, customer engagement, and decision-making.

This project follows a complete end-to-end data analytics workflow, starting from data cleaning to dashboard creation and insight generation.

---

## Project Objectives

- Clean and prepare raw customer transaction data using Python
- Store and analyze the data using SQL
- Identify trends in customer behavior, product performance, and purchase patterns
- Create interactive dashboards in Power BI
- Provide business insights and recommendations based on data analysis

---

## Tools and Technologies Used

- Python (Pandas, NumPy)
- PostgreSQL / MySQL
- Power BI
- Jupyter Notebook
- GitHub

---

## Dataset Information

The dataset contains customer shopping transactions with the following details:

- Customer information (Age, Gender, Location, Subscription Status)
- Purchase details (Product, Category, Amount, Season, Color, Size)
- Customer behavior (Discount usage, Purchase frequency, Review rating, Shipping type)
- Total Records: 3,900
- Total Columns: 18

---

## Project Workflow

### 1. Data Preparation (Python)

- Loaded dataset using pandas
- Handled missing values
- Renamed columns for consistency
- Created new features such as age_group and purchase_frequency_days
- Prepared cleaned data for analysis

### 2. Data Analysis (SQL)

- Loaded cleaned data into PostgreSQL
- Wrote SQL queries to analyze:
  - Revenue by gender
  - Customer segments
  - Product performance
  - Subscription impact
  - Purchase trends

### 3. Dashboard Creation (Power BI)

- Connected Power BI to SQL database
- Created interactive dashboards showing:
  - Sales overview
  - Customer analysis
  - Product performance
  - Purchase trends

### 4. Insights and Recommendations

Based on the analysis:

- Loyal customers contribute significantly to revenue
- Discounts influence purchase behavior
- Certain products and categories perform better than others
- Customer segmentation helps improve marketing strategies

## How to Run the Project

4. Connect Power BI to the database and open the dashboard file
5. Explore the dashboard and review insights
